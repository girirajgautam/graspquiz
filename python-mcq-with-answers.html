Python MCQ with Answers

Quick MCQ revision for exams and interviews.

  1. Which keyword defines a function? Answer: `def`
  2. Python is interpreted or compiled? Answer: Interpreted
  3. Which type is immutable? Answer: `tuple`
  4. Which symbol starts a comment? Answer: `#`
  5. Which function gets input? Answer: `input()`
  6. What does `len()` return? Answer: Number of elements
  7. Which keyword handles exceptions? Answer: `except`
  8. Dictionary keys must be? Answer: Hashable
  9. Which operator is exponent? Answer: `**`
  10. Which data type stores key-value pairs? Answer: `dict`
  11. What does `range(5)` produce? Answer: 0 to 4
  12. What is list slicing syntax? Answer: `[start:stop:step]`
  13. What does `append()` do? Answer: Adds one item at end
  14. What is `None` in Python? Answer: Null-like singleton object
  15. Which module gives math functions? Answer: `math`
  16. What is correct package installer? Answer: `pip`
  17. Which statement creates set? Answer: `set()` or `{1,2}`
  18. What does `pass` do? Answer: No operation placeholder
  19. What does `break` do? Answer: Exits loop
  20. What does `continue` do? Answer: Skips current iteration
